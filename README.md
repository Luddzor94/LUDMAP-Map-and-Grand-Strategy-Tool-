# LUDMAP-Map-and-Grand-Strategy-Tool-
In Browser html application. 
REQUIREMENTS: A BLACK AND WHITE IMAGE OF YOUR INTENDED MAP. (HARD REQUIREMENT!) /// A RGB PATTERN ON A BLACK BACKGROUND (COMPLETELY OPTIONAL, BUT GARGANTUAN TIME SAVER!)
both of the images i use personally are 4k, but i see no reason why smaller images should not suffice. just make sure both are PNGs. 

HOW TO USE: CLICK ON region_editor(2)html. A WINDOW SHOULD OPEN IN YOUR BROWSER. 
IN THE INPUT PANEL, PICK YOUR BLACK AND WHITE MAP AS THE PRIMARY PNG (MASK)
IN THE GENERATION PANEL, DECIDE ON HOW MANY REGIONS YOU WANT ( LAND SEED COUNT) AND THE DIMENSIONS OF THE OCEAN GRID (LARGER NUMBER = LESS GRIDS!)
/// NOTE: I RECOMMEND " BOX" AS THE GENERATION MODE ///

SOME OF THE PARAMETERS ARE HARDCODED INTO THE APPLICATION, BUT YOU CAN ADD YOUR OWN BY CLICKING ON "ADD DEFAULT PARAMETER"

IN ORDER TO MERGE REGIONS, CLICK ON A REGION, ENABLE MERGE MODE (LEFT HAND PANEL), AND CLICK ON THE REGION YOU WANT TO FUSE WITH. IT'S THAT SIMPLE.

YOU CAN ADD/REMOVE WHATEVER FACTION OR NATION YOU WANT IN THE CONTROL FACTION PANEL. WRITE WHATEVER YOU WANT AND CLICK " ADD FROM TEXT"

IN THE EXPORT PANEL, YOU CAN EXPORT A JSON FILE, OR 3 DIFFERENT VERSIONS OF THE MAP, DEPENDING ON YOUR NEEDS. 

HOW TO USE OPTIONAL PARAMETER MAP: PAINT A BLACK BACKGROUND IN WHATEVER COLOR YOU WANT ( you should be able to edit what color translates to which stats in the Parameter Map Panel!)
THE APPLICATION WILL READ THE RED, GREEN BLUE AND ALPHA VALUES AND DISTRIBUTE THOSE STATS ACROSS THE REGIONS. COLORS COMBINE SO PURE WHITE = ALL STATS MAXIMIZED.

EXAMPLE: YOU WANT ALL THE REGIONS IN THE EAST TO HAVE HIGH VALUES OF THE STAT "WEALTH". RATHER THAN EDITING 200 REGIONS FOR HAND, YOU SIMPLY PAINT RED ON THE EASTERN PART OF YOUR PARAMETER MAP, AND YOU ARE DONE. 

/// NOTE: MAKE SURE THAT YOUR PARAMETER MAP IS THE SAME DIMENSIONS AS YOUR PRIMARY PNG, OTHERWISE THEY MAY NOT LINE UP CORRECTLY.///

IN THEORY, THE EXPORTS CAN BE USED IN ANY GAME ENGINE TO CREATE A PARADOX-STYLE MAP. THE JSON HOLDS ALL THE STATS YOU APPLIED TO THE REGIONS.
